Projektbeskrivning:

My Wedding är en plattform för bröllopsplanering och nätverkande, utformad för blivande brudar och brudgummar för att hitta inspiration inför sin stora dag. Plattformen är också perfekt för bröllopsfotografer och florister, eftersom deras arbete är idealiskt för målgruppen för webbplatsen. Som fotograf kan du marknadsföra dig själv och hitta många nya kunder. Syftet med detta projekt var att skapa ett femte portfolioprojekt för Code Institutes program, Full Stack Software Developer. Webbplatsen är byggd med React, JSX, CSS och är ansluten till en backend-API byggd med Django Rest Framework. Applikationen innehåller full CRUD-funktionalitet för att skapa, uppdatera och ta bort inlägg och att-göra-uppgifter.

Genom att använda agile-metodik och verktyg som GitHub Projects har jag strukturerat och planerat projektet på ett effektivt sätt, vilket har lett till en smidig utvecklingsprocess. Genom att skapa detaljerade User Stories har jag säkerställt att varje del av webbplatsen uppfyller användarnas behov och förväntningar. Jag fokuserade på att skapa en ren och modern design med tydliga navigeringsmönster och användarvänliga gränssnitt.
